# Introduction to AI and Robotics

Welcome to the AI/Spec-Driven Book with Embedded RAG Chatbot. This educational resource focuses on ROS 2 as the middleware nervous system for humanoid robots, designed for AI students and developers entering humanoid robotics.

## About This Book

This book provides comprehensive coverage of ROS 2 concepts, communication patterns, and robot description formats (URDF) with a specific focus on humanoid robotics applications. Each module builds on fundamental concepts and provides practical examples using rclpy, the Python client library for ROS 2.

- **Module 1**: The Robotic Nervous System (ROS 2) - Core concepts, communication patterns, and URDF

Start with Module 1 to learn the fundamentals of ROS 2 and how it functions as a middleware nervous system for humanoid robots.
