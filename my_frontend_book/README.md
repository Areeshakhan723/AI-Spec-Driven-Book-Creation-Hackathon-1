# AI/Spec-Driven Book with Embedded RAG Chatbot

This repository contains an educational book about AI and robotics, specifically focusing on ROS 2 as the middleware nervous system for humanoid robots. The book is built with Docusaurus and includes interactive content for students and developers entering humanoid robotics.

## Features

- Comprehensive coverage of ROS 2 fundamentals for humanoid robotics
- Practical examples and tutorials
- Interactive content and exercises
- Embedded RAG chatbot for enhanced learning experience

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start
```

The documentation will be available at `http://localhost:3000`.

## Project Structure

- `docs/` - Documentation content in Markdown format
- `static/` - Static assets like images
- `src/` - Custom React components (if any)
- `specs/` - Project specifications and planning documents
